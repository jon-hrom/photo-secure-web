# Чеклист тестирования редактора коллажей

## Автоматический режим

### Выбор коллажей
- [ ] Открывается список коллажей для 1 фото
- [ ] Переключение на 2 фото показывает другие шаблоны
- [ ] Переключение на 3 фото показывает шаблоны с 3 слотами
- [ ] Клик по коллажу применяет его к текущему развороту
- [ ] Активный коллаж подсвечен фиолетовой рамкой

### Навигация
- [ ] Кнопка ← переключает на предыдущий разворот
- [ ] Кнопка → переключает на следующий разворот
- [ ] Кнопки отключаются на первом/последнем развороте
- [ ] Клик по миниатюре внизу переключает на разворот
- [ ] Активный разворот подсвечен в менеджере

### Автозаполнение
- [ ] Кнопка "Автозаполнение" заполняет все слоты фотографиями
- [ ] Фотографии распределяются последовательно
- [ ] Если фото больше чем слотов, остальные не используются

## Ручной режим

### Активация
- [ ] Кнопка "Ручной режим" переключает режим
- [ ] Кнопка меняет цвет на синий в активном состоянии
- [ ] Появляется панель инструментов сверху
- [ ] Появляется правая панель с фотографиями
- [ ] Показывается инструкция по использованию

### Выбор слота
- [ ] Клик по слоту выделяет его фиолетовой рамкой
- [ ] Появляются 4 фиолетовые точки по углам
- [ ] Появляются кнопки управления слотом

### Перемещение слота
- [ ] Клик и перетаскивание перемещает слот
- [ ] Слот не выходит за границы разворота
- [ ] Перемещение плавное и отзывчивое

### Изменение размера
- [ ] Перетаскивание верхнего левого угла изменяет размер
- [ ] Перетаскивание верхнего правого угла изменяет размер
- [ ] Перетаскивание нижнего левого угла изменяет размер
- [ ] Перетаскивание нижнего правого угла изменяет размер
- [ ] Минимальный размер слота 50x50px
- [ ] Курсор меняется при наведении на углы

### Управление слотами
- [ ] Кнопка "Добавить слот" создаёт новый слот
- [ ] Кнопка "Дублировать" копирует выбранный слот
- [ ] Кнопка "Очистить фото" удаляет фото из слота
- [ ] Кнопка "Удалить" удаляет слот полностью
- [ ] Кнопки показываются только при выборе слота

### Применение фотографий
- [ ] Клик по фото в правой панели выделяет его
- [ ] Повторный клик на выбранный слот применяет фото
- [ ] Фото масштабируется и обрезается по слоту
- [ ] Можно заменить фото в слоте

## Финальные проверки

### Визуал
- [ ] Развороты отображаются корректно
- [ ] Корешок книги виден по центру
- [ ] Безопасные отступы соблюдаются
- [ ] Миниатюры разворотов отображаются правильно

### Производительность
- [ ] Переключение между разворотами быстрое
- [ ] Перетаскивание слотов не лагает
- [ ] Изменение размера плавное

### Сохранение
- [ ] Кнопка "Завершить" сохраняет все развороты
- [ ] Данные передаются в onComplete корректно
- [ ] Можно вернуться назад кнопкой "←"
